Para obter a chave de API necessária para utilizar o **Google Places API**, siga os passos abaixo:

### Passo 1: Crie uma Conta no Google Cloud

1. Acesse o **Google Cloud Console**: [console.cloud.google.com](https://console.cloud.google.com/).
2. Se você ainda não tiver uma conta, crie uma conta do Google. Caso já tenha uma, faça login.
3. Após o login, você será direcionado ao painel de controle do Google Cloud.

### Passo 2: Crie um Novo Projeto

1. No painel do Google Cloud Console, clique em **Selecionar um Projeto** na parte superior da tela.
2. Clique em **Novo Projeto**.
3. Escolha um nome para o projeto e selecione a conta de cobrança (caso ainda não tenha uma conta de cobrança, você precisará configurar uma). Clique em **Criar**.

### Passo 3: Ativar o Google Places API

1. Após criar o projeto, vá para o menu de navegação à esquerda e clique em **APIs e Serviços** > **Biblioteca**.
2. Na página de **Biblioteca de APIs**, procure por **Places API**.
3. Clique em **Google Places API** e depois clique em **Ativar**.

### Passo 4: Criar uma Chave de API

1. Após ativar a API, vá para **APIs e Serviços** > **Credenciais**.
2. Clique em **Criar Credenciais** no topo da página e selecione **Chave de API**.
3. A chave de API será gerada e exibida na tela. Você pode copiá-la e usá-la no seu código.

### Passo 5: Configurar Restrições de Uso (Opcional)

É altamente recomendado restringir o uso da chave de API para evitar o uso indevido. Para configurar restrições:

1. Após gerar a chave, clique sobre ela na seção **Credenciais**.
2. Em **Restrições de chave**, você pode:
   - **Restringir o uso por IP** (para que apenas requisições de certos IPs possam usar a chave).
   - **Restringir as APIs** (limitando a chave para ser usada apenas com o **Google Places API** e outras APIs que você configurar).

### Passo 6: Usando a Chave de API no Código

Agora, substitua `'SUA_CHAVE_API'` no seu código pelo valor da chave de API gerada. Exemplo:

```python
import requests

def consulta_empresas(empresa_list):
    api_key = 'SUA_CHAVE_API'  # Substitua por sua chave gerada
    
    for nome_empresa in empresa_list:
        url = f'https://maps.googleapis.com/maps/api/place/textsearch/json?query={nome_empresa}&key={api_key}'
        response = requests.get(url)
        data = response.json()

        if 'results' in data:
            for result in data['results']:
                nome = result.get('name')
                telefone = result.get('formatted_phone_number', 'Telefone não disponível')
                print(f"Nome: {nome}, Telefone: {telefone}")
        else:
            print(f"Não encontrado: {nome_empresa}")

# Exemplo de lista de empresas
empresas = ["Restaurante Exemplo", "Loja ABC", "Escritório XYZ"]
consulta_empresas(empresas)
```

Agora, seu código deve ser capaz de consultar informações de empresas usando o **Google Places API**.

### Observação
A Google Places API tem **limites de requisições gratuitas** por mês (atualmente, 200 dólares em créditos gratuitos por mês). Após atingir esse limite, pode ser cobrado. Verifique a [documentação oficial](https://developers.google.com/maps/documentation/places/web-service/overview) para mais detalhes sobre as limitações e custos.
